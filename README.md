                                                  Smart Quiz Application

This is a Python-based quiz application where users can test their knowledge in Python, Artificial Intelligence, and Cyber Security.

The application asks multiple-choice questions and provides a final result based on the user's performance.


## About Project:

I created this project to practice Python programming concepts such as:

- Dictionaries
- Lists
- Functions
- Random module
- Time handling
- File handling

The quiz contains different categories and difficulty levels. Users can select a category and answer randomly selected questions within a given time limit.


## Features:

- Select quiz category
- Choose difficulty level (Easy / Hard)
- Random questions
- Time limit for answering questions
- Score calculation
- Percentage calculation
- Pass or Fail result
- Save quiz results in a file
